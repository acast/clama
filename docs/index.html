Local AI assistant for macOS and Ollama

Clama

Commands that stay on your Mac.

Clama understands natural phrases like open Safari, next track, what did I do?, and runs them locally with no cloud noise.

What it does well

Simple commands that work without extra context.

Opens and controls

Finder, Safari, Downloads, Documents, Pictures, Music, and system actions like volume.

Remembers simple things

Name, preferences, recent actions, and open apps persist between sessions.

Does not waste time

It runs through Ollama and local tools. More like a useful terminal assistant than a chat app.
