Локальный ассистент для macOS и Ollama

Clama

Команды, которые остаются на твоем Mac.

Clama понимает обычные фразы вроде открой Safari, следующий трек, что я делал? и выполняет их локально, без лишней болтовни и без облачного шума.

Open source on GitHub
Быстрый старт
npm link
clama "привет"
Clama

Белая витрина для светлого продукта и теплого локального ассистента.

Что делает хорошо

Простые команды, которые работают без лишнего контекста.

Открывает и управляет

Finder, Safari, Downloads, Documents, Pictures, Music и системные действия вроде громкости.

Помнит простое

Имя, предпочтения, последние действия и открытые приложения сохраняются между запусками.

Не тратит лишнее

Работает через Ollama и локальные инструменты. Это больше похоже на полезный терминальный ассистент, чем на чат.
